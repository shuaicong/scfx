## 2. 页面清单

| 页面 | 路由 | 原型文件 | 说明 |
|------|------|----------|------|
| 任务列表 | `/tasks` | task-list-prototype.html | 任务管理和筛选 |

### 3.3 创建任务表单

#### 3.3.1 任务信息
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | input | 是 | 最大50字符 |
| 数据源 | select | 是 | 粮信网/我的钢铁网/中华粮网/其他 |
| 任务描述 | textarea | 否 | 最大200字符 |

#### 3.3.2 触发配置
**单次触发**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 执行日期 | datepicker | 是 | 选择执行日期 |
| 执行时间 | timepicker | 是 | 时:分:秒 |

**周期触发**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 周期类型 | radio | 是 | 每天/每周/每月 |
| 触发时间 | timepicker | 是 | 时:分:秒 |
| 周期选项 | checkbox | 条件 | 每周:周一至周日 / 每月:1-31日 |
| 结束条件 | radio | 是 | 永不结束/指定日结束/重复次数结束 |

**Cron 表达式**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Cron表达式 | input | 是 | 支持5-8位格式 |
| 计算按钮 | button | - | 计算未来5次触发时间 |
| 结束条件 | radio | 是 | 同上 |

#### 3.3.3 脚本配置
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 脚本内容 | Monaco Editor | 是 | Python脚本，支持语法高亮 |

### 3.4 创建任务流程
```
1. 填写任务信息
      │
      ▼
2. 配置触发方式 ──────────────────┐
      │                          │
      ▼                          │
3. 编写脚本内容                   │
      │                          │
      ├─── 保存 ─────────────────┤
      │     │                    │
      │     ▼                    │
      │  创建初始版本 v1         │
      │     │                    │
      │     ▼                    │
      │  返回任务列表             │
      │                          │
      └─── 保存并执行 ───────────┘
            │
            ▼
       创建执行记录
            │
            ▼
       跳转到任务详情页
```

### 3.5 API 接口

#### POST /api/tasks
**请求体**
```json
{
  "name": "粮信网玉米晨报采集",
  "source": "liangxin",
  "description": "每日定时采集粮信网玉米价格数据",
  "triggerType": "cycle",
  "triggerConfig": {
    "type": "daily",
    "time": "08:00:00"
  },
  "scriptContent": "# Python script..."
}
```

**响应**
```json
{
  "code": 200,
  "data": {
    "id": 1,
    "name": "粮信网玉米晨报采集",
    "status": "enabled",
    "currentVersion": 1,
    "createdAt": "2026-04-30T12:00:00"
  }
}
```

### 4.3 统计卡片
| 卡片 | 数值 | 说明 |
|------|------|------|
| 全部 | 12 | 所有任务数量 |
| 启用 | 10 | 状态为enabled的任务 |
| 停用 | 2 | 状态为disabled的任务 |
| 成功 | 8 | 今日执行成功次数 |
| 失败 | 3 | 今日执行失败次数 |
| 今日 | 1 | 今日执行任务数 |

### 4.4 筛选功能
| 筛选 | 类型 | 选项 |
|------|------|------|
| 数据源 | select | 全部 / 粮信网 / 我的钢铁网 / 中华粮网 / 其他 |
| 状态 | select | 全部 / 启用 / 停用 |
| 搜索 | input | 按任务名称模糊搜索 |

### 4.5 列表操作
| 操作 | 按钮 | 说明 |
|------|------|------|
| 查看详情 | 点击行 或 [▸] | 跳转到任务详情页 |
| 启用/停用 | 状态toggle | 切换任务状态 |
| 批量选择 | ☐ | 多选任务 |
| 批量启用 | button | 批量启用选中的任务 |
| 批量停用 | button | 批量停用选中的任务 |
| 批量删除 | button | 批量删除选中的任务 |

### 4.6 API 接口

#### GET /api/tasks
**查询参数**
| 参数 | 类型 | 说明 |
|------|------|------|
| source | string | 数据源筛选 |
| status | string | enabled/disabled |
| keyword | string | 任务名称关键词 |

**响应**
```json
{
  "code": 200,
  "data": {
    "items": [
      {
        "id": 1,
        "name": "粮信网玉米晨报采集",
        "source": "liangxin",
        "triggerType": "cycle",
        "triggerDesc": "每日 08:00",
        "nextExecutionTime": "2026-05-01 08:00:00",
        "successRate": 0.98,
        "status": "enabled",
        "executionCount": 156
      }
    ],
    "total": 12
  }
}
```

#### POST /api/tasks/{id}/enable
启用任务

#### POST /api/tasks/{id}/disable
禁用任务

#### DELETE /api/tasks/{id}
删除任务

#### POST /api/tasks/batch-enable
批量启用
```json
{ "ids": [1, 2, 3] }
```

#### POST /api/tasks/batch-disable
批量禁用

#### DELETE /api/tasks/batch
批量删除

## 10. 状态说明

### 10.1 任务状态
| 状态 | 值 | 说明 |
|------|-----|------|
| 启用 | enabled | 任务正常运行，参与调度 |
| 停用 | disabled | 任务暂停，不参与调度 |

### 10.2 执行状态
| 状态 | 值 | 说明 |
|------|-----|------|
| 待执行 | pending | 等待执行 |
| 执行中 | running | 正在执行 |
| 成功 | success | 执行成功完成 |
| 失败 | failed | 执行异常 |
| 已取消 | cancelled | 用户手动取消 |

### 10.3 执行状态流转
```
pending ──> running ──> success
                  │
                  └──> failed
                  │
                  └──> cancelled
```
