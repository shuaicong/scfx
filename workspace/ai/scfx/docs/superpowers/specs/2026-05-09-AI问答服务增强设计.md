# AI 问答服务增强设计方案

## 1. 背景与目标

### 1.1 背景
- 现有 AI 问答服务已实现基础 RAG 问答功能
- 用户需要更丰富的回答形式：自动图表、可视化数据、多源引用
- 知识来源涵盖：网页、Word/PDF 文档、数据库数据

### 1.2 目标
- 流式响应：AI 回答逐字显示，打字机效果
- 图表自动渲染：识别可图表化的数据，自动渲染 ECharts
- 多源引用展示：区分网页/文档/数据库，支持跳转和预览

## 3. 核心功能

### 3.1 流式响应
- 后端 SSE 流式返回 AI 回答
- 前端逐字显示打字效果
- 支持中断（取消生成）

### 3.2 图表自动渲染
- AI 回答中嵌入 `<chart>` 标记
- 前端解析标记，调用 ECharts 渲染

**标记格式：**
```html
<chart type="line|bar|pie" title="图表标题"
       data='{"x":["label1","label2"],"y":[100,200]}'
       source="数据来源说明" />
```

**支持的图表类型：**
| 类型 | 用途 | 示例 |
|------|------|------|
| line | 时间序列趋势 | 玉米价格走势图 |
| bar | 分类对比 | 各产区价格对比 |
| pie | 占比展示 | 粮食品类占比 |

### 3.3 多源引用展示

| 来源类型 | 展示图标 | 交互方式 |
|----------|----------|----------|
| 网页 | 🌐 | 点击新窗口打开 |
| Word/PDF | 📄 | 弹窗预览 |
| 数据库 | 📊 | 显示关联图表或说明 |

## 4. AI 回答格式设计

### 4.1 完整回答示例

```
根据农业农村部和粮信网的数据：

【价格趋势】
近期玉米价格呈上涨趋势。
<chart type="line" title="全国玉米价格走势(元/吨)"
       data='{"dates":["5/26","5/27","5/28","5/29","5/30"],"prices":[2700,2715,2728,2740,2750]}'
       source="农业农村部监测数据" />

【区域对比】
东北产区价格高于全国平均水平：
<chart type="bar" title="主产区玉米价格对比"
       data='{"regions":["东北产区","华北产区","华中产区"],"prices":[2750,2680,2720]}'
       source="粮信网调研数据" />

【品类占比】
玉米在主要粮食品种中占比最高：
<chart type="pie" title="主要粮食品种占比"
       data='{"categories":["玉米","小麦","稻谷","大豆"],"values":[35,28,22,15]}'
       source="国家统计局" />

【数据来源】
- 🌐 [农业农村部价格监测](https://www.moa.gov.cn/price) - 官方数据
- 🌐 [粮信网玉米日报](https://liangxin.com/report/123) - 市场报价
- 📄 [2025年玉米市场分析报告.pdf](/api/files/report.pdf) - 分析文档
- 📊 数据库 - 实时价格数据
```

### 4.2 Prompt 设计

```python
# app/services/llm.py

SYSTEM_PROMPT = """你是粮食价格领域的专业分析助理，基于提供的参考材料回答问题。

【参考材料】
{context}

【问题】
{question}

【输出要求】
1. 仅基于参考材料回答
2. 当数据适合图表展示时，插入 <chart> 标记：
   - 折线图(line)：时间序列趋势数据
   - 柱状图(bar)：分类对比数据（如区域对比）
   - 饼图(pie)：占比数据（如品类占比）
3. chart 标记格式：
   <chart type="line|bar|pie" title="图表标题" data='{{"x":["标签"],"y":[数值]}}' source="数据来源" />
4. 在回答末尾列出【数据来源】，区分类型：
   - 🌐 [来源名称](URL) - 网页来源
   - 📄 [文档名称](/api/files/xxx.pdf) - 文档来源
   - 📊 数据库 - 数据来源说明
5. 如资料不足，说明"根据现有资料无法回答"
6. 回答简洁专业，突出关键数据"""

USER_PROMPT = """问题：{question}

请基于参考材料回答。"""
```

## 5. API 设计

### 5.1 流式问答接口

```
POST /api/chat/stream

Request:
Content-Type: application/json
{
  "question": "最近玉米价格趋势如何？",
  "top_k": 5
}

Response: text/event-stream

event: text
data: {"type": "text", "content": "根据"}

event: text
data: {"type": "text", "content": "农业"}

event: text
data: {"type": "chart", "chartType": "line", "title": "玉米价格趋势",
       "data": {"dates":["5/26","5/27"],"prices":[2700,2715]},
       "source": "粮信网"}

event: text
data: {"type": "source", "sources": [
  {"type": "web", "name": "粮信网", "url": "https://liangxin.com/report/123"},
  {"type": "doc", "name": "分析报告.pdf", "url": "/api/files/report.pdf"}
]}

event: done
data: {"type": "done"}
```

### 5.2 文档预览接口

```
GET /api/files/{file_id}

Response:
- Content-Type: application/pdf 或 application/vnd.openxmlformats-officedocument.wordprocessingml.document
- 文件流下载或在线预览
```

## 6. 前端设计

### 6.1 组件结构

```
src/views/ai-chat/
├── AiChat.vue              # 主页面
├── components/
│   ├── MessageContent.vue  # 消息内容解析（含图表渲染）
│   ├── ChartRenderer.vue   # ECharts 图表组件
│   ├── DocumentPreview.vue # 文档预览组件
│   └── SourceLink.vue      # 来源链接组件
```

### 6.2 MessageContent.vue 解析逻辑

```javascript
// 解析流程
1. 文本内容按 <chart> 标记分割
2. 文字部分 → HTML 渲染（支持加粗、链接等）
3. chart 标记 → ChartRenderer 组件
4. 来源链接 → SourceLink 组件
```

### 6.3 图表样式

**统一风格：**
- 深色背景（与聊天界面一致）
- 暖色调数据（#f5c87a 金色系）
- 网格线：半透明白色
- 字体：白色/浅灰

## 7. 技术选型

| 组件 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 图表库 | ECharts | 5.x | 功能全面，折线/柱/饼图支持好 |
| 流式传输 | SSE | - | Server-Sent Events，简单可靠 |
| 文档预览 | pdf.js | - | PDF 在线预览 |
| Word预览 | mammoth.js | - | Word 转 HTML 预览 |
| 实时流 | fetch + ReadableStream | - | 前端流式接收 |
